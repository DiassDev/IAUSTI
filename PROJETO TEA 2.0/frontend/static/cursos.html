O que é a lógica ?

Como pensamos logicamente ?

Lógica é a forma como organizamos o pensamento para entender, decidir ou resolver algo. É o jeito racional de pensar, observando o que faz sentido e o que não faz. Em outras palavras: a lógica é o caminho que seguimos para chegar a uma conclusão verdadeira.

Pensar logicamente é:

  • Observar uma situação com atenção.
  • Procurar as causas e as consequências.
  • Seguir passos claros, um de cada vez.
  • Evitar “pular etapas” ou se basear só no que “parece certo”.

Exemplo simples:

Se está chovendo, e eu sair sem guarda-chuva, eu fico molhado.

Logo: se está chovendo → devo levar o guarda-chuva.
